API Generator

API Generator est une application web Full-Stack qui permet de générer automatiquement des endpoints REST à partir de descriptions en langage naturel. L’utilisateur décrit simplement le comportement attendu d’une fonction, puis l’intelligence artificielle génère le code correspondant et crée un endpoint API exploitable immédiatement.

Le frontend, développé avec Next.js, offre une interface moderne, interactive et réactive permettant de créer, visualiser et tester facilement les fonctions générées. Ce projet met en avant l’intégration de l’IA dans le développement logiciel afin d’accélérer la création d’APIs et d’améliorer la productivité.
