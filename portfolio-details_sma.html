Description du projet

Ce projet consiste à développer un système multi-agent intelligent capable d’automatiser le processus de matching entre les CVs et les offres d’emploi. L’objectif est de réduire le temps d’analyse manuelle tout en améliorant la pertinence des résultats grâce à l’intelligence artificielle.

Le système utilise plusieurs agents spécialisés pour extraire les compétences des CVs, analyser les exigences des offres, comparer les profils et calculer un score de compatibilité. Cette approche permet de produire un classement clair des candidats et d’aider à la prise de décision dans le recrutement.

Le frontend a été développé avec Next.js, React et Tailwind CSS pour offrir une interface moderne, rapide et intuitive. Le backend repose sur Python et Flask, avec l’intégration de CrewAI et Groq LLM pour orchestrer les agents intelligents et générer des analyses avancées.

Fonctionnalités principales

Analyse des CVs

Extraction automatique des compétences, informations et éléments clés depuis les CVs.

Analyse des offres

Compréhension des exigences, technologies et profils recherchés dans les offres d’emploi.

Système Multi-Agent

Utilisation d’agents IA pour répartir les tâches d’analyse, de comparaison et de scoring.

Scoring intelligent

Calcul d’un score de compatibilité pour classer les candidats selon leur adéquation avec l’offre.
